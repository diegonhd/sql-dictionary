# Formação Completa em SQL: Do Zero à Administração de Sistemas

Este repositório contém a progressão completa de estudos em bancos de dados relacionais, cobrindo desde consultas básicas até a segurança e otimização de sistemas escaláveis. Como estudante de Engenharia de Computação, este material serve como uma base sólida para o desenvolvimento de software profissional.

## 🗺️ Mapa de Aprendizado (Módulos)

Abaixo, um resumo de cada etapa concluída:

| Módulo | Foco Principal | Conceitos Chave |
| :--- | :--- | :--- |
| **src0** | **Fundamentos** | Seleção de dados, filtros básicos (`WHERE`), ordenação e padrões de busca (`LIKE`). |
| **src1** | **Relacionamentos** | Junções de tabelas (`JOINs`), agrupamentos (`GROUP BY`), subconsultas e teoria dos conjuntos. |
| **src3** | **Design de Banco** | Evolução de schemas, restrições de integridade (`CHECK`, `UNIQUE`) e alteração de tabelas existentes. |
| **src4** | **Manipulação e Automação** | CRUD completo, importação de CSVs, automação com **Triggers** e abstração com **Views**. |
| **src5** | **Performance** | Otimização com **Índices**, manutenção de base (`VACUUM`) e controle de concorrência/transações. |
| **src6** | **Segurança e Adm** | Gestão de usuários/privilégios, defesa contra **SQL Injection** e transição para ambientes **MySQL**. |

---

## 🚀 Tecnologias e Ferramentas

* **SQLite**: Utilizado para aprendizado de lógica, manipulação de arquivos locais e testes rápidos.
* **MySQL**: Introduzido para cenários de arquitetura Cliente-Servidor e permissões granulares.
* **Data Cleaning**: Técnicas aplicadas para normalizar e limpar bases de dados reais.

## 🛠️ Como Navegar neste Repositório

Cada pasta `srcX` contém seu próprio `README.md` detalhado com a explicação de cada script SQL e o banco de dados correspondente necessário para a execução.
